# IndianArtMate Project

## Overview
IndianArtMate is a project designed to showcase and promote Indian art and culture through a digital platform. The application allows users to explore various forms of art, connect with artists, and engage with the community.

## Features
- **Art Gallery**: Browse through a collection of artworks from various artists.
- **Artist Profiles**: Learn more about individual artists and their works.
- **Community Engagement**: Participate in discussions and events related to Indian art.
- **Upload Artworks**: Artists can upload their own artworks to share with the community.

## Installation
To get started with the IndianArtMate project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/IndianArtMate.git
   ```

2. Navigate to the project directory:
   ```bash
   cd IndianArtMate
   ```

3. Install the necessary dependencies for both the backend and frontend:
   - For the backend:
     ```bash
     cd Backend
     npm install
     ```
   - For the frontend:
     ```bash
     cd ../Frontend
     npm install
     ```

## Usage
To run the application, follow these steps:

1. Start the backend server:
   ```bash
   cd Backend
   node server.js
   ```

2. In a new terminal, start the frontend development server:
   ```bash
   cd Frontend
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:3000` to view the application.
