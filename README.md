# IndianArtMate Project

## Description
IndianArtMate is a comprehensive application designed to showcase and manage various art pieces. It integrates a backend server, a frontend interface, and Python scripts for data processing and machine learning.

## Technologies Used
- Node.js
- Express.js
- React
- Python
- MongoDB

## Installation Instructions
1. Clone the repository.
2. Navigate to the Backend directory and run:
   ```bash
   npm install
   ```
3. Navigate to the Frontend directory and run:
   ```bash
   npm install
   ```

## Usage Instructions
- Start the backend server:
  ```bash
  npm run server
  ```
- Start the frontend application:
  ```bash
  npm run dev
  ```
