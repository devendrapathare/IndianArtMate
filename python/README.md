# Python Scripts for IndianArtMate

## Overview
This directory contains Python scripts used for data processing and machine learning tasks related to the IndianArtMate project.

## Key Scripts
- **data_preprocessing.py**: Handles data cleaning and preparation for analysis.
- **model_training.py**: Contains logic for training machine learning models.
- **predict.py**: Used for making predictions based on the trained models.
- **sentiment_routes.py**: Provides API routes for sentiment analysis.

## Running the Scripts
To start the python server follow these steps:
```bash
    cd python
    python app2.py
```
Make sure to install the required dependencies before running the scripts.
